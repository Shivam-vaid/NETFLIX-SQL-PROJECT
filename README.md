# Netflix SQL Data Analysis Project

This project analyzes Netflix movies and TV shows data using SQL to generate meaningful business insights. The project includes data cleaning, filtering, aggregation, and analytical queries to understand content trends, ratings, genres, and release patterns.

## Tools Used

* SQL
* MySQL
* GitHub

## SQL Concepts Covered

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* Aggregate Functions
* Subqueries
* Window Functions

## Key Insights

* Total movies and TV shows available
* Most common genres and ratings
* Top countries producing Netflix content
* Content trends by release year

## Skills Demonstrated

* Data Analysis
* SQL Query Writing
* Data Cleaning
* Business Insights
